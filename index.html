SRToolbox

SRToolbox is based on the idea of chummer-sr5. First we wanted to create "chumpy", but after a while we decided to create a much bigger project. We want to make Shadowrun great again! (SCNR). We want to make it easy for everyone to create a character, to master a Shadowrun game, to create a Toolbox for your own P'n'P. Stay tuned!
